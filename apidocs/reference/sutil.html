Sutil Namespace

Type/Module Description

Bind

Bindings for observables and the DOM. For example, use an IObservable<bool> to toggle an element's class attribute

BindArray

Bindings for array observables and the DOM. For example, IObservable<int[]>. The Bind class already handles lists. BindArray exists to eliminate compilation errors related to overloading of the each method.
