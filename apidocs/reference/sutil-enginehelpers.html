Ev

Full Usage: Ev

Returns: SutilEventEngine

DOM event handlers. Strictly speaking, these are attributes, but it's useful for them to have their own namespace. For example 

 Html.button [
     Ev.onClick (fun _ -> console("click"))
 ]

Returns: SutilEventEngine
