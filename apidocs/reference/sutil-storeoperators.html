s |-> f

Full Usage: s |-> f

Parameters:

Alias for `Store.getMap`, takes a store and applies a mapping function then returns the value from the evaluated function

This might be called foldMap

s : IStore<'a>
f : 'a -> 'b
Returns: 'b
Example

let store: IStore<{| name: string; budget: decimal |}> = Store.make {| name = "Frank"; budget = 547863.26M |} let formattedBudget: string = store |-> (fun model -> sprintf $"$ %0.00M{model.budget}") printf %"Budget available: {formattedBudget}
